# Amazon Sales Data Cleaning Project

## Project Objective
This project focuses on data cleaning and preprocessing of an Amazon sales dataset using Python and Pandas.

The goal was to:
- Understand dataset structure
- Identify missing values and data quality issues
- Clean and transform data
- Perform feature engineering
- Generate an analysis-ready dataset

## Data Cleaning Steps Performed
- Removed missing values
- Converted price columns into numeric format
- Cleaned percentage values
- Converted ratings into float datatype
- Created new feature: `discount_amount`
- Saved cleaned dataset
