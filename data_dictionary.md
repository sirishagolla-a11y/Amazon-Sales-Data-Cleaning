# Amazon Sales Dataset - Data Dictionary

| Column Name | Description | Data Type | Business Relevance |
|-------------|-------------|------------|--------------------|
| product_id | Unique identifier for each product | String | Helps track products uniquely |
| product_name | Name of the product | String | Product identification |
| category | Product category | String | Used for category analysis |
| discounted_price | Current selling price | Float | Helps analyze current pricing |
| actual_price | Original product price | Float | Used to compare discounts |
| discount_percentage | Percentage discount offered | Float | Helps analyze discount strategy |
| rating | Product rating given by users | Float | Measures customer satisfaction |
| rating_count | Number of ratings received | String | Measures product popularity |
| about_product | Product description | String | Product information |
| user_id | Unique user identifier | String | Tracks users |
| user_name | Name of reviewer | String | Reviewer identification |
| review_id | Unique review identifier | String | Tracks reviews |
| review_title | Title of review | String | Review summary |
| review_content | Detailed review content | String | Customer feedback analysis |
| img_link | Product image URL | String | Product display |
| product_link | Amazon product URL | String | Product navigation |
| discount_amount | Difference between actual and discounted price | Float | Measures savings offered |
